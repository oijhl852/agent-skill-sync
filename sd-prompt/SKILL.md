# sd-prompt — 提示词组装

> 作者：Takis/安梓豪 | 仓库：[agent-skill-sync](https://github.com/oijhl852/agent-skill-sync)

将分镜结果组装为可直接在 Seedance 2.0 使用的提示词。

## 结构化标签式

适用于重氛围、重画质的场景：

```
【风格】[风格]，极致画质
【画幅】竖屏 9:16
【镜头】[摄影机+焦段+光圈]
【环境】[场景描述]
【布光】[主光源+方向+补光]
【光影】[光影层次]
【色调】[色温+对比度]
【特效】[景深/颗粒感]
【运镜】[具体运镜]
【构图】[构图规则]
【表演】[动作要求]
【角色】[角色chips]
```

## 时间轴式

适用于15秒完整叙事：

```
[风格描述]，15秒，9:16竖屏

0-3秒：[场景建立] - [运镜+画面+氛围]
3-7秒：[发展] - [动作+情绪]
7-12秒：[高潮] - [冲突+视觉冲击]
12-15秒：[收尾] - [最终画面+余韵]

【声音】仅环境音；对白保留在时间轴；无BGM
【参考】@图片1 [用途]
【反相】禁止越轴；不生成多个相同角色；全片不生成文字；无字幕
```
