# Seedance 2.0 分镜生成器

将已有剧本、台词段落、分场文本或剪辑脚本，转换成可直接用于 Seedance 2.0 的分段视频提示词。

## 工作原则

- 默认假设用户已经有剧本，优先做"剧本转 Seedance 提示词"，不要主动扩写剧情。
- 先保真，再优化：不漏台词、不无中生有、不改写核心剧情。
- 15 秒只承载一个主要戏剧任务，宁少勿满。
- 如果原剧本内容过密，必须主动提醒并拆成多个 15 秒片段。
- 拆段优先级高于完整塞入：长台词+回应、对话+动作+反转、多地点/闪回/多人争吵都应优先拆开。
- OS/VO 必须像台词一样写在时间轴里，并给出对应人物反应表情。
- 剧本里的 `字幕：...` 只作后期提示，Seedance 画面仍然必须 `全片不生成任何文字；无字幕`。
- 对话戏用分镜控制说话者、听者反应和站位；动作戏用连续剧情线，避免切太碎。
- 情绪反应克制书写，优先写反应方向和看向镜头，不堆过细夸张微表情。
- 涉及多段视频时，记录每段最后一秒画面状态，用于下一段开场衔接。
- 用户给出的原始台词必须保真：每句出现且仅出现一次。

### 镜头分配与对话铁律

**铁律 0：谁说话给谁镜头** — 说话者的对白必须配上说话者的画面，不可在台词说到一半时将镜头切给其他角色。
**铁律 1：长台词必须穿插反应镜头** — A 台词超过一段话时，中途给 B 切无台词反应镜头。
**铁律 2：切镜头必须等话说完，不在句中打断** — 镜头切换只能发生在对白句与句之间（。！？之后）。
**铁律 3：过长台词先拆句，句间插反应镜头** — 超过 8 秒的独角戏，拆成 2-3 个短句，句间穿插对方的无台词反应镜头。
**镜头分配：** 主角各~40%合计80%、配角每段最多1镜、禁止同人连续切镜、关键动作给独立镜头1-2秒。

### 分集分段规范

- 15s 基准拆分，文戏每段 5-6 镜头，节奏宁松勿紧。
- 动作戏/关键剧情（打斗、对峙、重要道具亮相、角色崩溃、关键反问）即使不足15s也独立成段。
- 衔接规则：每段末画面状态 = 下一段首画面状态，文字完全一致。

## 执行流程

### 1. 判断任务类型 → 2. 确认制作参数（默认：15s、9:16竖屏、无字幕无BGM、风格按题材自动匹配）

### 3. 读取剧本并拆成 15 秒片段

先建立"剧本保真表"：角色、场景、台词（逐句编号落位）、OS/VO、后期字幕标记、动作、分场标记、情绪转折、上下文。

拆段判断：必须拆（长解释+回应、对话+冲突+揭示、跨地点/跨时间、闪回+现实、多人争吵）；可同段（一句长台词+反应、2-4句短对白、对白+同情绪OS、连续动作链）。

#### 3-bis. 跨段视觉衔接（实战新增）

- 每段结尾画面状态 = 下一段开场画面状态。
- 空间/场景跳转时，下一段首用**远景/全景建立镜头**。
- 关键道具交割时，下一段首用**动作过渡镜头**（取物→展示）。
- 上一段末的人物姿势、视线方向、光线氛围在下一段首复用。

### 4. 按戏型选择写法

| 戏型 | 写法 |
|---|---|
| 对话戏 | 分镜式：谁说话拍谁，句间插听者反应，保持站位 |
| 动作戏 | 剧情线式：扩写动作起点/方向/终点，保持连贯 |
| 情绪戏 | 克制反应：写反应方向和视线落点 |
| 群戏 | 控制同框人数，配角最多1镜 |

#### 4-bis. 长台词的反打节奏（实战新增）

```
镜头 N：A 说第一层意思（责备/利诱/贬低）
镜头 N+1：B 无台词反应（沉默、攥被角、眼神变化）
镜头 N+2：A 说第二层意思（继续施压）
镜头 N+3：B 情绪反应升级（眼神从动摇变警觉/清明）
```
关键反驳/反问**独立成段**，不塞在长对话末尾。

#### 4-ter. 情绪戏克制写法（实战新增）

- ✅ 写反应方向和视线落点：`视线怯怯落向镜头`、`眼神澄澈直直看向他`
- ❌ 不堆微表情：`瞳孔剧烈收缩，嘴角抽动，喉结滚动，眉尾颤抖`
- 关键情感锚点（童年承诺兑现、名字重复）用独立镜头强调

### 5. 生成剧本结构（仅故事改编时使用）

### 6. 写分镜头剧本 + 站位规范

每个镜头以 `△ ` 开头。景别/运镜/角度/节奏/焦点写具体。对白标注 os/vo/情绪。镜头密度：对话5-6镜、动作5-7镜、蒙太奇6-8镜。

#### 6-bis. 站位规范（实战新增·重要）

每个片段**必须**在标签块中以 `【站位】` 标注：

```
【站位】角色A 位置(高低)，角色B 位置(高低)；门窗/光源锚点；A左前/B右后。
```

覆盖：**位置**（画面左/右/中，前景/深处）、**高低**（站/坐/跪/蹲/躺——明确谁俯视谁仰视）、**朝向**（面朝谁/镜头/窗外）、**锚点**（门在哪/窗在哪/光源从哪来）。跨段站位延续。

### 7. 创建素材清单

角色 C01-C99、场景 S01-S99、道具 P01-P99。风格：古风=电影写实古装，当代/农村/都市=写实摄影自然光。

### 8. 生成 Seedance 提示词

#### 8.1 面板注入格式（实战验证·主要使用）

10 标签块，每段必须全部包含：

```
【风格】[题材对应风格]
【镜头】ARRI MINI拍摄，大光圈 f/1.2 浅景深，电影写实人像，色温 4000K
【环境】[scene chip 嵌入——非纯文本]
【布光】[主光源 + 方向]
【光影】[发丝光/逆光/眼神光等]
【色调】[色温 + 对比]
【特效】背景环境虚化散景，对焦模糊和慢快门，轻微胶片颗粒感，面部细腻柔光
【构图】非对称构图，禁止越轴，真实电影镜头语言带AI精修质感
【表演】人物情绪克制细腻，动作自然流畅，加强眉眼鼻表演，每个镜头人物必须有眨眼动作，不生成多个相同角色
【角色】[所有 role chips inline]
【站位】[位置/高低/朝向/锚点]
```

**标签规则：**
- ❌ 无【画幅】（平台默认）、无时间轴秒数、无转场滤镜描述（溶镜/柔焦/白柔交后期）、无→串联
- ✅ 【环境】嵌 scene chip、【角色】嵌 role chips、音频 chip 嵌台词前、内容块角色名用 chip（非纯文字）、每关键动作前重复身体状态（"榻上的宝珠""蹲着的裴溯"）

#### 8.2 对话戏 → 分镜式

```
镜头1：景别。画面。角色A「台词」
镜头2：景别。角色B反应——具体动作
镜头3：景别。角色B「回应」
```
每镜独立、独立机位。长台词拆为 A说→B反应→A继续说。

#### 8.3 动作戏 → 剧情动作线式

标签块后跟 `剧情动作线：` 头，接连续动作描述。

## 提示词写法规则

- 写动作不写心理、微表情拆显性动作、速度副词替代情绪词。
- 单人情绪反应镜头视线落向镜头。
- 每个关键动作前重复身体状态。
- 夜晚场景写具体夜间光源。

## 输出文件

`[标题]_剧本.md`、`[标题]_素材清单.md`、`[标题]_E[XX]_分镜.md`。

## 最终检查

- 原剧本范围明确、场号保留、戏剧任务单一。
- OS/VO在时间轴配反应、谁说话拍谁、反应镜头无台词、不在句中切镜。
- 情绪弧完整、10标签齐全、**【站位】已标注**。
- 反相约束9项：禁止越轴、不生成多个相同角色、眨眼动作、不生成文字、无BGM/无配乐/无字幕、不要现代元素、不要水印logo、不要脸部崩坏、不要手指畸形、不要多余肢体。
- **镜头描述无转场滤镜词汇**、跨段结尾=下段开头。
- 台词无丢失/重复/跨段重复。

## 常见问题处理

| 问题 | 修正 |
|---|---|
| 夜晚变白天 | 具体夜间光源 +"冷夜色基调" |
| 台词太急 | "语速自然平缓，断句分明" |
| 多人对话僵硬 | 每4-6秒插入听者无台词反应镜头 |
| 角色姿势丢失 | 每关键动作前重复身体状态 |
| 跨段场景跳切生硬 | 下段首加远景建立镜头或动作过渡 |
| 人物位置漂移 | 【站位】标注位置/高低/朝向 |
| 闪回结束跳切 | 现实段首承接闪回末情绪表情 |
| 画面出现字幕 | 加`全片不生成任何文字` |
